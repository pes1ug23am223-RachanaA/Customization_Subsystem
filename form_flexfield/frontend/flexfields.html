Segments —

Each segment is a discrete data component of this flexfield. For a KEY flexfield, segments form a structured code (e.g. DEPT-LEVEL-GRADE). For DESCRIPTIVE, each segment stores a named extra attribute.

Segments define the structure of the flexfield value. Changes take effect immediately.
