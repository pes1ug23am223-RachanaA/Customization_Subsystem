Trigger Appraisal Workflow Workflow Integration

Start a performance appraisal workflow for an employee. Choose a predefined workflow template or create a custom workflow. Integrated with Workflow Engine via IPerformanceIntegration.

Standard Performance Appraisal (6 steps) Self Assessment → Manager Review → Peer Feedback → HR Evaluation → Final Rating → Acknowledgement
Fast Track Review (3 steps) Self Assessment → Manager Review → Confirmation
Probation Review (4 steps) Self Assessment → Line Manager → HR Review → Decision

Create Performance Form Form Designer Integration

Create an appraisal form linked to a performance cycle. Choose a predefined form template with standard fields, or start from scratch in the Form Designer.

Standard Appraisal Form (8 fields) Self Rating, Manager Rating, Goals Achieved, Strengths, Areas to Improve, Training Needs, Overall Rating, Comments
KPI Review Form (5 fields) KPI Name, Target, Actual Achievement, Score, Manager Remarks
Probation Completion Form (6 fields) Employee Name, Probation Period, Performance Rating, Recommendation, Manager Signature, HR Notes
